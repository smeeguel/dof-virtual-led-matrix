Quick-start Summary

  1. If needed, install both DOF builds (x64 + x86) into C:\DirectOutput.
  2. Unzip the provided package anywhere (example: C:\vPin\VirtualLED).
  3. Back up and replace DirectOutput.dll in both x64 and x86 folders.
  4. Copy sample config files into C:\DirectOutput\Config (or contact for merge help if already customized).
  5. Run the virtual matrix .exe before launching VPX or front-end tables.
